# 鲜花商城系统 python68

## 项目概述


## 技术栈

- **编程语言**：Python 3.12.4
- **数据库**：MySQL 8.0
- **Web框架**：Django 5.0.6
- **前端技术**：HTML5, CSS3, JavaScript, jQuery

## 功能模块

### 1. 系统角色

- **用户**：支持登录注册，密码修改，图形验证码，RSA加密传输。

### 2. 系统功能

- **首页**：展示分类鲜花数据，包含轮播Banner和图片懒加载。
- **详情页面**：展示商品具体信息，支持加入购物车、收藏等功能。
- **分类**：对鲜花分类展示，支持排序分页。
- **会员中心**：
 - 个人信息管理：修改头像，鲜花推荐。
 - 收货地址管理：增删改地址，设置默认地址。
 - 订单管理：概览，状态过滤，自动取消未支付订单。
 - 评论晒单：对商品进行评价和打分。
 - 收藏管理：展示和管理收藏的鲜花。
- **购物车**：管理用户加入的购物车数据，生成订单。
- **订单**：添加署名和留言祝福，选择支付方式。
- **花语**：展示花语列表，详情，相关文章推荐。
- **实时私聊**：提供实时双向沟通，支持文本、图片、表情。

### 3. 运行环境

- 确保Python环境已安装，并配置好MySQL数据库。
